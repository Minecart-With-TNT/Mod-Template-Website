# Mod Authors

A comma-separated list of author names displayed in the in-game mod list and on distribution platforms like Modrinth and CurseForge.

```
ExampleDev, AnotherDev, ThirdDev
```

These values are written into `fabric.mod.json` (Fabric) or `neoforge.mods.toml` (NeoForge / Multiloader) and shown to players in the mod menu.
